# Rewrite Patterns

Use these patterns when rewriting. Preserve meaning and useful detail.

## From Quoted Labels To Natural Speech

```text
Before: 它不只是停在“聊天框”里回答问题。
After: 它不只是停在聊天框里回答问题。
```

Variants:

```text
Before: 从“代码助手”变成“AI 员工”。
After: 从代码助手往 AI 员工走了一步。

Before: 以前更多是在“给我建议”，现在开始能“替我动手”。
After: 以前更多是给我建议，现在开始能替我动手。

Before: 我会把它放在“需要真实账号”的场景里。
After: 我会把它放在需要真实账号的场景里。
```

## From Rhetorical Dashes To Spoken Rhythm

```text
Before: Browser — 写完页面自己打开验收
After: Browser，写完页面自己打开验收
```

Variants:

```text
Before: 这个插件很适合一类人——经常做前端的人。
After: 这个插件很适合经常做前端的人。

Before: 我试完只有一个感觉——装完以后，它真的不一样。
After: 我试完以后感觉很明显，装完以后，它真的不一样。

Before: 它解决的是一个问题——汇报材料。
After: 它解决的是汇报材料这件事。
```

## From Over-Polished To More Spoken

```text
Before: 这三个装完，Codex 的边界会明显往外扩一圈。
After: 这三个装完以后，我明显感觉 Codex 能做的事多了一截。
```

Variants:

```text
Before: 这个插件适合想快速做一个小工具并上线的人。
After: 如果你想快速做个小工具，还想顺手上线，可以先试这个。

Before: 它把这些零碎步骤串起来了。
After: 它能把中间这些零碎步骤一起跑掉。

Before: 这类场景可以放到同一个工作流里处理。
After: 这类事可以放在一起让它处理。
```

## From Correcting The Reader To Sharing Observation

```text
Before: 很多人觉得 X，但其实是 Y。
After: 我这段时间观察到，Y 更值得关注。
```

Variants:

```text
我自己试下来，Y 对结果影响更明显。
实际操作里，Y 往往更容易决定结果。
这次我最明显的感受是：Y 不能跳过。
```

## From Authority Judgment To Experience Judgment

```text
Before: 真正重要的不是 X，而是 Y。
After: 我自己试下来，Y 对结果的影响更明显。
```

Variants:

```text
这件事我后来才发现，Y 比 X 更容易影响结果。
我现在会优先看 Y，再看 X。
```

## From Speaking For The Reader To Giving A Path

```text
Before: 你可能不知道该从哪里开始。
After: 如果刚开始做，可以先从这三步开始。
```

Variants:

```text
如果现在还没思路，可以先试一个最小版本。
可以不用一上来做完整流程，先把第一步跑通。
```

## From Teaching "Ordinary People" To Offering A Starting Point

```text
Before: 这才是普通人真正能开始的地方。
After: 如果现在不知道从哪里开始，可以先选一件每天重复做的小事。
```

Variants:

```text
这个起点不大，但很容易跑通。
先从一个低成本的小任务开始，会比一次性研究很多工具更稳。
```

## From Report-like Analysis To Lived Reaction

```text
Before: 这个方法的核心是减少执行成本。
After: 我用下来最直接的感受是：它把最耗时间的执行部分先接过去了。
```

Variants:

```text
这一步让我明显轻松了一点。
这个结果比我预期更可用。
这里我反而有点犹豫，因为...
```

## From Generic Evaluation To A Specific Place

```text
Before: 整体优化效果很明显。
After: 开头那三句改完以后，听起来不再像在教育观众。
```

Variants:

```text
第二段的说教感少了。
最后一句的姿态更平了。
这里从“纠正读者”变成了“分享经验”。
```

## Do Not Overdo It

Avoid making every sentence start with "I". The goal is not more first-person language; the goal is a more equal relationship between writer and reader.

Avoid replacing all clear judgments with weak language. Strong opinions are allowed when they come from evidence, experience, or transparent reasoning.
